# 移交收件箱 (Handover Inbox)

请将来自情报工作区 (`D:\Intel_Briefing`) 的 **任务报告 (Mission Reports)** 或 **项目计划书 (Project Plans)** 投递至此处。

## 代理指令 (Instructions for Agents)
1. 当此目录下出现新文件时，读取文件以理解项目范围。
2. 使用 `process_handover` 工作流在 `projects/` 目录下启动项目。

## 文件命名规范
- `[项目名称]_MissionPlan.md`
- `[YYYY-MM-DD]_Strategy_Report.md`
